## Standard Rubric Structures

### 3-Point Rubric (Open Response Questions)
| Score | Criteria |
|-------|----------|
| 3 | Complete mechanism with evidence AND reasoning |
| 2 | Partial mechanism OR incomplete reasoning |
| 1 | Minimal response, major gaps |
| 0 | No response or off-topic |

### 4-Point Rubric (Integration Questions)
| Score | Criteria |
|-------|----------|
| 4 | Uses multiple evidence types with explicit reasoning |
| 3 | Uses one evidence type well |
| 2 | Mentions evidence but doesn't connect |
| 1 | Vague response |
| 0 | No response |

### 5-Point Rubric (SEP Tasks)
| Score | Criteria |
|-------|----------|
| 5 | Complete with multiple evidence types, explicit reasoning chain |
| 4 | Mostly complete, minor gaps in reasoning |
| 3 | Correct direction but incomplete mechanism |
| 2 | Significant gaps, partial understanding |
| 1 | Major misconceptions evident |
| 0 | No response |

### 7-Point Rubric (Highest-Value Questions)
| Score | Criteria |
|-------|----------|
| 7 | Complete mechanism: [all required elements] |
| 5-6 | Most elements present |
| 3-4 | Partial mechanism |
| 1-2 | Minimal elements |
| 0 | No response |
